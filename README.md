# Spring Framework Projects

## 📌 Overview

This repository contains my Spring Framework practice projects. These projects cover the core concepts of Spring, including Dependency Injection (DI), Inversion of Control (IoC), Bean Configuration, Autowiring, Stereotype Annotations, Java-Based Configuration, Lombok integration, and Loose Coupling.

The repository is part of my Java and Spring learning journey and is intended for interview preparation and hands-on practice.
